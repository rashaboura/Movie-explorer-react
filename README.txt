Movie Explorer (React)
=================================
React rewrite that preserves your existing styling and UX.

Quick start
-----------
1) Install Node LTS (>=18).

2) In this folder, run:
   npm install
   echo "VITE_TMDB_KEY=YOUR_TMDB_API_KEY" > .env
   npm run dev

3) Build for deployment:
   npm run build

4) Deploy the 'dist' folder to Netlify / Vercel / GitHub Pages.
